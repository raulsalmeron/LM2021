Crea un script que pida al usuario que escriba por pantalla un texto. Transforme el texto en un array de letras minúsculas (para facilitar la solución del ejercicio). Recorra todo el array en búsqueda de cada tipo de vocal a, e, i, o, u. Recorra todo el array contabilizando espacios en blanco. Finalmente el script debe escribir por pantallas todos y cada uno de los textos y devolver el texto tratado según la imagen a continuación.
